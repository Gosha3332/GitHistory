Эмммммммм... ну короче... эммммм... ну это — сервис для отслеживания истории активности репозитория на GitHub, в общем чтобы удобно статистику было смотреть.

Стек технологий:
- бекенд — С#;
- фронтенд — html, css.

Фронтенд будет отправлять запросы на локальный сервер, а тот в свою очередь через апишку гихаба будет брать данные репозитория, обрабтывать их, возвращать обратно.

Создается экземпляр класса репозитория с его данными и внутри уже будут методы по обработке данных, которые и будут отправлятся на фронтенд.

Будет реализовано:
- Поле ввода;
- Страница со статой (график зависимости количества коммитов по дням, сколько звезд, владелец, язык рограммирования и т.д).
